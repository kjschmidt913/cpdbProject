Accusals Network

Each edge represents an accusal in which the connected officers nodes are listed together on. Larger nodes denote supervisors. Each color represents a different police unit. There are 98 units. Upon hovering over a node, you can also see officer ID, unit, and complaint percentile. Scroll around the network to explore! When the nodes stop moving, the data has fully loaded. This is a lot of data and may take a few minutes to load.
